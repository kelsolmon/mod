Cuestionario Sobre el contenido de la Unidad I.

1. Qué es un Sistema Operativo

Un Sistema Operativo (SO) es un conjunto de programas que gestionan los recursos de hardware y software de un ordenador y facilitan su funcionamiento.

2. Componentes de un Sistema Operativo

Los componentes de un Sistema Operativo incluyen el kernel (núcleo), el gestor de procesos, el gestor de memoria, el sistema de archivos, los controladores de dispositivos, la interfaz de usuario y las utilidades del sistema.

3. Tareas del Sistema Operativo

Las tareas del Sistema Operativo son gestionar los recursos del sistema (CPU, memoria, dispositivos de entrada/salida), administrar los procesos y la memoria, proporcionar una interfaz para la interacción del usuario, y garantizar la seguridad y la integridad del sistema.

4. Funciones de un Sistema Operativo

Las funciones de un Sistema Operativo incluyen la gestión de recursos, la ejecución y finalización de programas, el control de acceso a los recursos del sistema, la administración de archivos y directorios, y la gestión de la memoria.

5. Características de un Sistema Operativo

Las características de un Sistema Operativo pueden incluir multitarea, multiusuario, tiempo compartido, portabilidad, capacidad de red, seguridad, y facilidad de uso.

6. Tipos de Sistema Operativo

Los tipos de Sistema Operativo incluyen Windows, macOS, Linux (y sus distribuciones), Unix, Android, iOS, entre otros.

7. Sistemas Operativos Para PC

Para PC, los Sistemas Operativos más comunes son Windows, macOS y diversas distribuciones de Linux.

8. Sistemas Operativos Móviles

Para dispositivos móviles, los Sistemas Operativos más utilizados son Android e iOS.

9. Sistema Operativo según el Ordenador y el Tipo de Aplicaciones

El Sistema Operativo varía según el tipo de ordenador (PC, móvil, servidor) y el tipo de aplicaciones que se ejecuten (por ejemplo, de propósito general, científicas, embebidas).

10. Compatibilidad de los Sistema Operativo

La compatibilidad entre los Sistemas Operativos se refiere a la capacidad de ejecutar programas diseñados para un SO en otro, o de compartir archivos entre ellos.

11. Cuál es la función básica de un Sistema Operativo

La función básica de un Sistema Operativo es gestionar los recursos de hardware y software de un sistema informático para que los programas puedan ejecutarse y los usuarios puedan interactuar con el sistema de manera eficiente.

12. Mencione tres Sistema Operativo más utilizados

Los tres Sistemas Operativos más utilizados son Windows, macOS y Linux.

13. Menciona algunas tareas que realizan los Sistemas Operativos

Algunas tareas que realizan los Sistemas Operativos incluyen la gestión de memoria, la planificación y ejecución de procesos, la gestión de archivos y dispositivos, la administración de redes y la seguridad del sistema

14. Qué es un sistema operativo multitareas

Un sistema operativo multitareas es aquel que permite la ejecución simultánea de múltiples programas.

15. Qué es un sistema operativo multiusuario

Un sistema operativo multiusuario es aquel que permite que varios usuarios accedan al sistema y utilicen sus recursos de manera concurrente.

16. Cómo funciona un sistema operativo

Un Sistema Operativo funciona como un intermediario entre el hardware y el software, gestionando recursos, ejecutando programas y proporcionando una interfaz para que los usuarios interactúen con el sistema.

17. Cómo se utiliza un sistema operativo en un entorno gráfico

En un entorno gráfico, un Sistema Operativo se utiliza mediante interfaces gráficas de usuario (GUI), donde los usuarios interactúan con el sistema seleccionando y haciendo clic en objetos en la pantalla.

18. Cuándo inicia el funcionamiento del Sistema Operativo

El funcionamiento del Sistema Operativo inicia cuando se completa el proceso de inicio, que ocurre después de que se enciende o reinicia el ordenador.

19. Mencione las dos interfaces que forman parte del Sistema Operativo

Las dos interfaces que forman parte del Sistema Operativo son la interfaz gráfica de usuario (GUI) y la interfaz de línea de comandos (CLI).

20. Qué es un proceso en entorno de los sistemas operativos

Un proceso en el entorno de los Sistemas Operativos es cada programa que se ejecuta en el sistema, ya sea una aplicación o un servicio en segundo plano.

21. Qué es un sistema operativo distribuido y centralizado

Un Sistema Operativo Distribuido permite utilizar recursos de múltiples computadoras simultáneamente, mientras que un Sistema Operativo Centralizado permite utilizar recursos de una sola computadora.

22. Mencione los sistemas de archivos utilizados por los Sistemas Operativos

Algunos sistemas de archivos utilizados por los Sistemas Operativos incluyen NTFS y FAT32 en Windows, y diversos sistemas de archivos en Linux como ext4.

23. Mencione otras de las características actuales de los sistemas operativos

Otras características actuales de los Sistemas Operativos incluyen un amplio conjunto de controladores de dispositivos, virtualización, seguridad mejorada, actualizaciones automáticas y soporte para hardware y software modernos.

24. Que permite la exigencia de muchos controladores en un sistema operativo.

La existencia de muchos controladores en un Sistema Operativo permite una rápida compatibilidad con una amplia gama de dispositivos de hardware.
